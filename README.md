# yfskyline/dotfiles

## setup
cd  
git clone https://github.com/yfskyline/dotfiles.git  
cd ./dotfiles/  
sh ./dotfilesLink.sh

## macOS Setup
This assumes macOS monterey(12).
```shell
# Sign in to AppleID
$ open /System/Applications/System Preferences.app

$ cd mac
$ sh setup_mac.sh

$ open /Applications/Karabiner-Elements.app

```
